# Resort Management System

## Overview
This is a backend microservices-based Resort Management System built using **Java and Spring Boot**.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven
- REST APIs

## Services (Planned / Implemented)
- Booking Service
- Inventory Service
- Payment Service
- Billing Service
- API Gateway

## How to Run
1. Clone the repository
2. Configure `application.properties`
3. Run the Spring Boot application
